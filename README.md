# Shopping Website Automation Project

## Description
This project is an end-to-end automation framework for a shopping website using Java and Selenium.
It demonstrates hands-on experience in building robust automation frameworks, including Page Object Model (POM), Fluent, Factory Method, and Data-Driven Testing (DDT).

## Features
Automated test scripts for common shopping scenarios (login, register, search, add to cart, checkout).
Uses Selenium WebDriver with Java.
Implements fluent wait for stable test execution.
Modular framework design for maintainability.
Supports DTT with Jason.
#Technologies / Tools Used
Java,Selenium WebDriver,Maven,TestNG,Git & GitHub, GitHub actions
